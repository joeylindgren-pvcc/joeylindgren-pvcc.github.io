Operating Systems Terminology

Term Definition
Operating System The software that controls the overall operation of the computer
Job Queue The jobs that reside in mass storage waiting for execution
FIFO Abbreviation for "first in first out"
Interactive Processing New operating systems that were developed to allow a program being executed to carry on a dialogue with the user through remote terminals
Time-Sharing Operating systems that provided service to multiple users at the same time
Multiprogramming When time is divided into intervals and then the execution of each job is restricted to only one interval at a time
Application Software Consists of the programs for performing tasks particular to the machines utilization
System Software Performs those tasks that are common to computer systems in general
Utility Software Consists of software units that extend the capabilities of the operating system
Graphical User Interface (GUI) An interface in which objects to be manipulated, such as files and programs, are represented pictorially on the display as icons
Shell Older user interface that communicated with users through textual messages using a keyboard and a monitor screen
Kernel The internal part of an operating system
Device Driver The software units that communicate with the controllers to carry out operations on the peripheral deevices attached to the machine
File Manager A unit whose job is to coordinate the use of the machine's mass storage facilities
Memory Manager A component of an operating system's kernel which is charged with the task of coordinating the machine's use of main memory
Virtual Memory A large "fictional" memory space created by paging
Boot Strapping A procedure that is performed by a computer each time it is turned on. It is this procedure that transfers the operating system from mass storage
ROM (read-only memory) A small portion of a computer's main memory where the CPU expects to find its inital program is contructed from special nonvolatile memory cells. It is called ROM because its contents can be read but not altered
Deadlock The condition in which two or more processes are blocked from progressing because each is waiting for a resource that is allocated to another
Spooling A technique of holding data for output at a later but more convenient time
Super User (or administator) A person that can establish accounts, who gains highly priveleged access to the operating system by identifying him- or herself as the administrator during the login procedure. Once this contact is established, they can alter settings within the operating system
Multi-Factor Authentication A solution to the problem of weak passwords to secure computer accounts, is to require more than one piece of evidence to gain access
