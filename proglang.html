Chapter 6: Programming Languages

Term Definition
Machine Language sequences of instructions that are encoded as numeric digits
Debugging a tedious task that involves finding errors that must be located and corrected in machine languages
Assembler a program that was developed to convert mnemonic expressions into machine language instructions
Assembly Language a mnemonic system for representing programs
Translator a program that was written to translate programs expressed in high-level primitives into machine-language programs
Compiler similar to second-generation assemblers, except that it often had to compile several machine instructions into short sequences to simulate the activity requested by a single high-level primitive
Interpreter emerged as a nother means of implementing third-generation languages. These programs were similar to translators except that they executed the intstructions as they were translated instead of recording the translated version for future use
Cross-platform Software software that is independent of an operating system's design as well as the machine's hardware design and is therefore executable throughout a network
Natural Language languages like English, German, and Latin
Formal Language programming languages
Machine Independence software that is not dependent on the properties of a particular machine, and can therefore be used on any machine
High-level Programming Language a programming language with strong abstraction from the details of the computer
Object-Oriented Programming (OOP) a computer programming model that organizes software design around data, or objects, rather than functions and logic
Scripting Languages a subset of imperative programming languages
Data Types:
  • Integer
  • Float
  • String

  • numeric data that might contain values other than whole numbers, probably stored in floating point notation
  • a data type used in programming, that is used to represent text rather than numbers
Primitive Data Type data types that are included as primitives in a programming language, such as int for integer and char for character
Data Structure the conceptual shape or arrangement of data
Python Collection Data Structures:
  • List
  • Tuple
  • Set
  • Dictionary
  • used to store multiple items in a single variable, created using square brackets
  • used to store multiple items in a single variable, collection which is ordered and written with round brackets
  • used to store multiple items in a single variable, a collection which is unordered, unchangeable, and unindexed
  • used to store data values in key:value pairs, a colecction which is ordered, changeable, and do not allow duplicates
Function a set of instructions for performing a task that can be used as an abstract tool by other program units
Local Variable a variable declared within a function, can be referenced only within that function
Global Variable variables whose scopes are not restricted to a particular part of a program
Source Program the program in its original form
Object Program the translated version of the program
