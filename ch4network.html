Networking and The Internet

Term Definition
Network Classifications
  • LAN
  • MAN
  • WAN
 computer networks often classified as being either PAN, LAN, MAN, and WAN
  • (local area network) a collection of computers in a single building or building complex
  • (metropolitan area network) a network of intermediate size, such as one spanning a local community
  • (wide area network) links machines over a greater distance--perhaps in neighboring cities or on opposite sizes of the world
Network Models
  • Bus network
  • Star networks
 a database model that is designed as a flexible approach to representing objects and their relationships
  • a local area network(LAN) topography in which each node is connected to a main cable or link called a bus
  • a local area network(LAN) topography in which all nodes are directly connected to a common central computer(HUB)
Protocols
  • CSMA/CD Ethernet
  • CSMA/CA Wifi
 a set of rules for formatting and processing data
  • a medium access control method used notably in early Ethernet technology for local area netorking
  • a network multiple access method in which carrier sensing is used, but nodes attempt to avoid collisions by beginning transmission only after the channel is sensed to be "idle"
Combining Networks
  • Repeater
  • Bridge
  • Switch
  • Router
  • Gateway
 done by connecting the networks to form an extended communication system
  • a device that passes signals back and forther between the two original buses without considering the meaning of the signals
  • a device that connects two buses, but it looks at the destination address that accompanies each message accross the connection only when that message is destined for a computer on the other side
  • a bridge with multiple connections, allowing it to connect several buses rather than just two
  • the connection between networks to form an internet
  • the "point" at which one network is linked to an internet
Client/server model
  • p2p: peer-to-peer
  • Distributed System
  • Cloud Computing
 defines the basic roles player by the processes as either a client, which makes requests of other processes, or a server, which satisfies the requests made by clients
  • involves processes that provide service to and receive service from each other
  • consist of software units that execute as processes on different computers
  • huge pools of shared computers on the network can be allocated for use by clients as needed
Internet Addressing
  • Tier 1
  • Tier 2
  • Tier 3
 made up of a network address and a host(or local) address
  • consist of very high speed, high capacity international WANs. These are thought of as the backbone of the Internet
  • tend to be more regional in scope and less potent in their capabilities. These tend to be operated by companies in the communications business
  • an independent Internet, sometimes called an intranet, operated by a single authority that is in the business of supplying Internet access to indivdual home and businesses
IP Address
  • ICANN
  • DNS
  • TLD: Top-level Domain
  • Country-code TLDs
 an internet-wide addressing system that assigns a unique identifying address to each computer in the system
  • a non-profit corporation established to coordinate the Internet's operation
  • a Internet-wide directory system that are essentially directories that provide translation services to clients
  • used to reflect the domain's classification, which in this case is "educational", as indicated by the .edu suffix. Other examples consist of .gov for US government institutions, .org for non-profit organizations, and .com for commercial institutions
  • represents a specific geographical location and is the most efficient way to show search engines and users which country or region you're based in
Internet Protocols
  • FTP
  • HTTP
  • SMTP
 a set of rules for routing and addressing packets of data so that they can travel across networks and arrive at the correct destination
  • an application for accessing another computer from a great distance
  • (Hyper Text Transfer Protocol) as webserveres and browsers have become more sophisticated, more and more of these traditional network applications must be handled by webpages through https
  • defines a way that two computers on the network may interact when transmitting an email message from one host to the other
World Wide Web
  • URL
  • Client-side Activities
  • Server-side Activities
 had its origins in the work of Tim Berners-Lee, who realized the potential of combining internet technology with the concept of linked documents
  • a unique address give to each document to make it easer to locate and retrieve them
  • additional activity by either the browser or the webserver performed by a client
  • additional activity by either the browser or the webserver performed by a server
Internet Four-Layer Software
  • Application Layer
  • Transport Layer
  • Network Layer
  • Link Layer
 the framework for organizing the set of communication protocols used in the Internet and similar computer networks according to functional criteria
  • provides process-to-process data exchange for applications
  • handles host to host communications
  • provides internetworking between independent networks
  • contains communication methods for data that remains within a single network segment
Internet is:
  • Packet-swithing Network
  • Fault Tolerant
  • Redundant
 a vast network that connects computers all over the world
  • the transfer of small pieces of data across various networks
  • refers to the ability of a system to continue operating without interruption when one or more of its components fail
  • code that are in your source code, but are not required or have no logical effect to the output
Cybersecurity
  • Malware
  • Virus
  • Worm
  • Trojan Horse
  • Spyware
  • Phishing
  • DDoS Attack
 the study of protecting computers, networks, and data from such attacks
  • cyber attacks that incorporate the use of malicious software
  • software that infects a computer by inserting itself into programs that already reside in the machine. Then, when the "host" program is executed, the virus is also executed
  • an autonomous program that transfers itself through a network, taking up residence in computers and forwarding copies of itself to other computers
  • a program that enters a computer system disguised as a desirable program, such as a game or useful utility package, that is willingly imported by the victim
  • software that collects information about activities at the computer on which it resides and reports that information back to the instigator of the attack
  • a technique of obtaining information explicitly by simply asking for it
  • the work of generating the attacks is distributed over many hosts
Protection
  • Firewall
  • Spam Filter
 the process of incorporating protection principles and promoting meaningful access, safety and dignity
  • a computer network security system that restricts internet traffic in to, out of, or within a private network
  • algorithms that detect unsolicited, undesired, or infected emails, and block those messages from reaching inboxes
